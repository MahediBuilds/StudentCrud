## ⚙️ API Endpoints

| Method | Endpoint              | Description               |
|--------|-----------------------|---------------------------|
| POST   | `/students`           | Add a new student         |
| GET    | `/students`           | Get all students          |
| GET    | `/students/{id}`      | Get student by ID         |
| PUT    | `/students/{id}`      | Update student by ID      |
| DELETE | `/students/{id}`      | Delete student by ID      |
